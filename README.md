# Discord-Fake-Deafen

A lightweight BetterDiscord plugin that simulates being deafened while still letting you hear audio locally. Useful for focus, testing, or keeping audio feedback while appearing deafened to others.

## Features

- Toggle fake-deafen on/off from the BetterDiscord plugin list
- Simple toggle in UI next to settings button
- Minimal overhead and no external dependencies

## Install

1. Download the plugin file `Discord-Fake-Deafen.plugin.js`.
2. Open BetterDiscord settings.
3. Go to Plugins.
4. Click "Open Plugins Folder".
5. Move the plugin file into that folder.
6. Enable the plugin in the Plugins list.

## Usage

- Enable the plugin to add the button to the UI.
- Click the button to toggle fake-deafen on/off.
- Disable the plugin to return to remove from UI.

## Notes

- This plugin does not mute your local audio, it does ensure your local audio is muted prior, discord settings still apply.
- Behavior may change with Discord or BetterDiscord updates.
- Source code is also included if the user wishes to use or modify it locally.

## Troubleshooting

- If the plugin does not appear, confirm it is in the correct Plugins folder.
- If it fails to load, try reloading BetterDiscord or Discord.
